A application that is capable of simulating and fitting sub 1ns temporal-width laser pulses.
Fitting algorithms:
    Cubic spline
        dependencies: None
    Gaussian parametrization
        dependencies: None
    N-order Spline
        dependencies: None
    Levenberg Marquardt
        dependences: 
        https://github.com/pngts/Nonlinear-parameter-estimation-in-thermodynamic-models/blob/master/LevenbergMarquardt.m
        https://www.mathworks.com/matlabcentral/fileexchange/53440-jacobian-toolbox


Have fun simulating waveforms!
